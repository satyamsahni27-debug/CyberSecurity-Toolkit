# CyberSecurity Toolkit 🔐

A beginner-friendly cybersecurity project built using Python.

## 📂 Tools Included

- 🔎 Port Scanner
- 🔑 Password Strength Checker
- 🎣 Phishing URL Detector

## 🚀 How to Run

Clone the repository:

git clone https://github.com/satyamsahni27-debug/CyberSecurity-Toolkit.git

Then run any file:

python port_scanner.py
python password_checker.py
python phishing_detector.py
